dev10x-git

Git workflow utilities for clean repos, readable history, and parallel workspaces.

$ curl -LsSf https://brave-labs.github.io/dev10x-git/install.sh | sh
Fish Bash Zsh

What's Included

Branch Hygiene

gpb
Sync base branches, prune stale locals. The "morning routine".
gf
Verbose fetch with prune — see what moved on the remote.
stale-branches
List remote branches sorted by last commit date.

Worktree Management

gw [name]
Switch to a worktree with automatic context display.
gwa <name>
Add worktree with smart base detection and safety checks.
gwr [name]
Remove worktree + branch with confirmation and protection.

Rebase & Autosquash

grda
Non-interactive autosquash rebase onto origin/develop.
grma
Non-interactive autosquash rebase onto origin/main.
grd
Interactive rebase onto origin/develop.
gpf
Safe force push with --force-with-lease.

Cleanup

clean-orig
Remove .orig merge conflict artifacts.
clean-pyc
Remove compiled Python bytecode files.

Installation

The installer auto-detects your shell and places files in the right locations:

# Auto-detect shell
curl -LsSf https://brave-labs.github.io/dev10x-git/install.sh | sh

# Install for a specific shell
curl -LsSf https://brave-labs.github.io/dev10x-git/install.sh | sh -s fish
curl -LsSf https://brave-labs.github.io/dev10x-git/install.sh | sh -s bash
curl -LsSf https://brave-labs.github.io/dev10x-git/install.sh | sh -s zsh

# Install for all shells
curl -LsSf https://brave-labs.github.io/dev10x-git/install.sh | sh -s all
# Files installed to:
~/.config/fish/conf.d/dev10x-git.fish    # functions + aliases
~/.config/fish/completions/gw.fish       # tab completions
~/.config/fish/completions/gwa.fish
~/.config/fish/completions/gwr.fish

# Open a new terminal tab to activate.
# Files installed to:
~/.dev10x-git/git-utilities.bash         # functions + aliases
~/.dev10x-git/git-completions.bash       # tab completions

# Source lines added to ~/.bashrc automatically.
# Run: source ~/.bashrc
# Files installed to:
~/.dev10x-git/git-utilities.zsh          # functions + aliases
~/.zsh/completions/_gw                   # tab completions
~/.zsh/completions/_gwa
~/.zsh/completions/_gwr

# Source + fpath lines added to ~/.zshrc automatically.
# Run: exec zsh
